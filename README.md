## Sobre o repositório

Este repositório reúne exercícios práticos desenvolvidos em Python com foco em lógica de programação, estruturas condicionais, laços de repetição e operações matemáticas fundamentais.

## Conteúdo

Os exercícios abordam temas como:

- Verificação de valores dentro de intervalos
- Cálculo de médias escolares e análise de presença
- Estruturas de repetição com condição de parada
- Verificação de números perfeitos
- Conversão de números binários para decimal
- Classificação por faixa etária
- Simulação de reajuste salarial
- Cálculo de média de idades
- Divisão por subtrações sucessivas

## Objetivo

Praticar conceitos essenciais da programação em Python, incluindo:

- Variáveis e tipos de dados
- Entrada e saída de dados
- Operadores matemáticos e relacionais
- Estruturas condicionais
- Estruturas de repetição
- Acumuladores e contadores
- Resolução de problemas com lógica

## Tecnologias utilizadas

- Python 3
- Visual Studio Code
